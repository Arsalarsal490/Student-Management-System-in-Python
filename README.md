# Student Management System in Python

## Overview

This project is a console-based Student Management System developed in Python. It uses file handling to store student records in a text file and NumPy for statistical calculations. The system provides a simple menu-driven interface for managing student information.

## Features

- Add Student
- Display All Students
- Search Student by ID
- Update Student Record
- Delete Student Record
- Calculate Average Marks
- Display Highest Marks
- Display Lowest Marks
- Input Validation
- Duplicate Student ID Check
- File Handling using `students.txt`

## Technologies Used

- Python 3
- NumPy
- File Handling

## Project Structure

## How to Run

1. Install Python 3.
2. Install NumPy:

```bash
pip install numpy
```

3. Run the program:

```bash
python student_management_system.py
```

## Input Validation

- Student ID must contain only numbers.
- Name must contain only alphabets.
- Age must be a positive integer.
- Course must contain only alphabets.
- Marks must be between 0 and 100.
